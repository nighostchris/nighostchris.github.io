UST Freshman Chatbot( COMP3111 Project )

Our team implemented this LINE chatbot in the notorious course COMP3111. That's the first time I need to learn new framework in a pretty short time and implement the features from scratch. At last we got quite a number of expected features done despite the stone-age like standard IO interface restricted by LINE. Users can ask about the bus ETA, contact information of departments staffs and even rough estimation of time to go from one place to another within campus.
